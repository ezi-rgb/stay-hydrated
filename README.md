# Stay Hydrated

>**drink some water bro...**

Are you a human? Do you have functioning kidneys? Then you need this.

## Features

- **In Your Face:** Pops up dead center on your screen. You can't miss it.
- **Meme-Powered:** Uses animated GIFs because static text is for boomers.
- **Zero Bullshit:** No login, no tracking, no "subscribe to premium hydration". Just water.
- **Customizable Nagging:** Change the interval to whatever you want (default is 30 mins, but you can set it to 5 if you're a houseplant).

## How to Run

2. Install the magic image library:

5. Drink the water
